Absolute Positioning

This container is positioned relatively.

Relative positioning means I can still use margin:auto to center it, but then offset the box from the default position in the browser, in this case the offset is 100px from the top of the page.

This container is a child of the red div and has the folowing css style applied:

width:200px; height:200px; position:absolute; top:100px; right: 10px; border:1px solid green;

This stickman is constructed using divs positioned absolutely relative to the stick.
